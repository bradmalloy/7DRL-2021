Miner

Extracts resources from the ground. Build with "m"

Loader

Moves resources from A to B. Select with "l", then choose an input facing with "n", "s", "e", or "w". The output is always the opposite side. Press "Esc" to cancel building.

Conveyor Belt

Slides resources in one direction. Select with "c", then choose an output direction with "n", "s", "e", or "w". Belts don't have an input port - they can only receive items from other belts or loaders. Press "Esc" to cancel building.

Small Box

Stores resources. Build with "s". Can accept input from any direction, and can be accessed from any direction.
